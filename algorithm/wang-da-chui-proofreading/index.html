王大锤校对文稿

Posted by 北风 Blog on Saturday, September 5, 2020

TOC

题目来源：牛客网/字节跳动面试题

题目描述：

我叫王大锤，是一家出版社的编辑。我负责校对投稿来的英文稿件，这份工作非常烦人，因为每天都要去修正无数的拼写错误。但是，优秀的人总能在平凡的工作中发现真理。我发现一个发现拼写错误的捷径：

  1. 三个同样的字母连在一起，一定是拼写错误，去掉一个的就好啦：比如 helllo -> hello
  2. 两对一样的字母（AABB 型）连在一起，一定是拼写错误，去掉第二对的一个字母就好啦：比如 helloo -> hello
  3. 上面的规则优先“从左到右”匹配，即如果是 AABBCC，虽然 AABB 和 BBCC 都是错误拼写，应该优先考虑修复 AABB，结果为 AABCC

我特喵是个天才！我在蓝翔学过挖掘机和程序设计，按照这个原理写了一个自动校对器，工作效率从此起飞。用不了多久，我就会出任 CEO，当上董事长，迎娶白富美，走上人生巅峰，想想都有点小激动呢！

……

万万没想到，我被开除了，临走时老板对我说： “做人做事要兢兢业业、勤勤恳恳、本本分分，人要是行，干一行行一行。一行行行行行；要是不行，干一行不行一行，一行不行行行不行。” 我现在整个人红红火火恍恍惚惚的……

请听题：请实现大锤的自动校对程序

输入描述:

第一行包括一个数字 N，表示本次用例包括多少个待校验的字符串。

后面跟随 N 行，每行为一个待校验的字符串。

输出描述:

N 行，每行包括一个被修复后的字符串。

输入例子:

2 helloo wooooooow

输出例子:

hello woow

题目解析：

由题可知，一共有两种需要删除的类型：AABB -> AAB 和 AAA -> AA，这里我们需要考虑到 AAABB 的情况，由 AAA -> AA 得到 AABB，此时还需要再次进行删除得到 AAB，也就是说这种情况下进行了两次删除操作。

了解了需要删除的条件，我们从左到右依次遍历即可，因为需要删除的最长的情况为 AABB，有 4 个字符，我们需要三个变量 A、B、C 来保存前三个字符，每次与当前字符进行比较，观察是否满足 AAA 或 AABB 的格式，，满足则删除，不满足则将当前字符添加到结果集，并且更新 A、B、C。

需注意，当满足删除条件进行删除时不需要更新 A、B、C 变量，仅增加遍历的下标进行下一次比较即可，因为当前字符已经被删除了。

按照思路可以写出下面的代码：

public List<String> check2(List<String> strs) {
        List<String> res = new ArrayList<>();
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < strs.size(); i++) {
            sb.setLength(0);
            char[] chs = strs.get(i).toCharArray();
            char one = 0;
            char two = 0;
            char three = 0;
            for (int j = 0; j < chs.length; j++) {
                if (two != 0 && one == two && one == chs[j]) {
                    continue;
                } else if (three != 0 && three == two && one == chs[j]) {
                    continue;
                }
                sb.append(chs[j]);
                three = two;
                two = one;
                one = chs[j];
            }
            res.add(sb.toString());
        }
        return res;
    }
